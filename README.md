## Input examples:
### POST http://127.0.0.1:5000/user/login
```JSON
  {
    "authorization": {
      "name": "Morty",
      "password": "229"
    }
  }
```

### POST http://127.0.0.1:5000/user/new
```JSON
  {
    "authorization": {
      "name": "Morty",
      "password": "229",
      "confirm_password": "229"
    }
  }
```

### POST http://127.0.0.1:5000/post/new
```JSON
  {
    "post": {
      "slug": "i-am-in-space",
      "creator": "Morty",
      "title": "Get me out of here, RICK!1"
    }
  }
```
